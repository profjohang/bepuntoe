Cómo retomar el trabajo mañana?
Para volver exactamente a donde te quedaste:

Abre VS Code.

Ve a File > Open Folder y selecciona tu carpeta proyecto_be.

Abre una terminal nueva (Ctrl + Shift + ñ).

Liberar los permisos de ejecución: Set-ExecutionPolicy Unrestricted -Scope Process

Activa el venv: venv\Scripts\activate

Lanza el servidor: uvicorn main:app --reload

Verificar la rama: git branch

Preparar los archivos: 
git add . 
(Esto le dice a Git: "Toma todos mis cambios y ponlos en la sala de espera").

Guardar el progreso (Commit): 
git commit -m "Descripción de lo que hiciste" 
(Ejemplo: git commit -m "UI: Ajuste de colores institucionales en b.e").

git commit -m "feat: Ajustes de metodo para lograr register desde login, ajuste apariencia register" 

Sincronizar con la nube (Push): 
git push origin feature/mejoras-be

Confirmar que el árbol de trabajo está limpio ("working tree clean")
git status

Crear una nueva rama
git checkout -b feature/"nombre de la rama"
git checkout -b feature/mejoras-be

Detener el servidor: Presiona
Ctrl + C en la terminal. 
(Esto apaga el proceso de Uvicorn de forma segura).

Actualizar lista de librerías: 
pip freeze > requirements.txt 
(Muy importante: guarda la "receta" de instalación por si cambias de PC).

Desactivar el entorno virtual: 
deactivate

Cerrar la carpeta: 
Ve a File > Close Folder en VS Code.
